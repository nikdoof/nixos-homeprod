    hs-01: Headscale VPN Controller

    hs-01 is a NixOS microVM (CID 10) on hyp-01, running on VLAN 106 with a publicly routed IP. It runs Headscale — the self-hosted Tailscale control server — and an embedded DERP relay.

    Network

    Property Value
    Platform microVM, CID 10, VLAN 106
    IPv4 217.169.25.13/29
    IPv6 2001:8b0:bd9:106::13/64
    Gateway 217.169.25.9 (gw hosted VLAN)

    Open ports: 80, 443 (nginx → headscale), 3478 (STUN/DTLS for DERP).

    Headscale

    The Headscale instance at https://hs.doofnet.uk manages the Tailscale mesh network:

    Setting Value
    Server URL https://hs.doofnet.uk:443
    Base domain ts.doofnet.uk
    Magic DNS Enabled
    DERP region ID 999, "St. Helens"
    OIDC issuer https://id.doofnet.uk (Pocket-ID)
    Prefixes 100.64.0.0/10, fd7a:115c:a1e0::/48

    DNS configuration

    Magic DNS is enabled with split DNS:

    • doofnet.uk and incognitus.net resolve to internal DNS servers (10.101.1.2/3)
    • Global fallback uses Cloudflare (1.1.1.1, 1.0.0.1)
    • Search domains: ts.doofnet.uk, int.doofnet.uk, doofnet.uk

    ACL policy

    ACLs are defined as a JSON file and loaded by Headscale. Groups: home, vpn, admin. Tag-based routing allows selective access between groups.

    nginx reverse proxy

    nginx terminates TLS for hs.doofnet.uk (ACME via Let's Encrypt) and proxies to Headscale on 127.0.0.1:8080. Long timeout (3600s) is set for DERP WebSocket connections.

    Persistence

    Headscale data is bind-mounted from /persist/headscale to /var/lib/headscale.
